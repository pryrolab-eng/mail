### Issue: Campaigns module not showing

### Issue: Templates not saving
**Solution:** Check database connection and RLS policies

### Issue: Verification not working

### Issue: Sequences not displaying
**Solution:** Create a campaign first, then add sequences

### Issue: Build errors
**Solution:** Run `npm install` and restart dev server
